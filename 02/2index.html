Основні теги html

<h1> - представляє собою найважливіший заголовок першого рівня.

Приклад використання:
<h1>Заголовок першого рівня</h1>

Результат:

Заголовок першого рівня

<h6> - представляє собою найменш важливий заголовок шостого рівня.

Приклад використання:
<h6>Заголовок шостого рівня</h6>

Результат:

Заголовок першого рівня

<strong> - тег призначений для того щоб акцентувати увагу на тексті, браузери відображають такий текст жирним шрифтом.

Приклад використання:
<strong>Важливий текст</strong>

Результат:
Важливий текст

<em> - тег призначений для того щоб акцентувати увагу на тексті, браузери відображають такий текст курсивом.

Приклад використання:
<em>Виділенний текст</em>

Результат: Виділенний текст

<small> - тег змінює розмір шрифта на одиницю в порівнянні з основним текстом.

Приклад використання:
<small>Зменшенний текст</small>

Результат: Зменшенний текст

<ins> - тег призначений для виділення тексту, котрий було додано в нову версію документа, браузер відображає такий текст як підкресленний

Приклад використання:
<ins>Підкресленний текст текст</ins>

Результат: Підкресленний текст

<del> - використовується для виділення тексту, який було видалено в новій версії документа, браузер зазвичай помічає такий текст як закреслений.

Приклад використання:
<del>Закреслений текст</del>

Результат:
Закреслений текст

<sub> - відображає шрифт у вигляді нижнього індексу. Текст при цьому розташовується нижче за базову лінію усіх інших символів і має меньший розмір

Приклад використання:
<sub>Текст в нижньому індексі</sub>

Результат: Текст в нижньому індексі

<sup> - відображає шрифт у вигляді верхнього індексу. Текст при цьому розташовується вище за базову лінію усіх інших символів і має меньший розмір.

Приклад використання:
<sup>Текст у верхньому індексі</sup>

Результат:
Текст у верхньому індексі

<code> - тег призначений для відображення однієї або декількох рядків тексту, які представялють собою програмний код, браузер зазвичай відображає вміст даного контейнера як зменшений моноширний текст.

Приклад використання:
<code>Текст програмного коду html</code>

Результат: Текст програмного коду html

<kbd> - використовується для позначення текста котрий набираєтся з клавіатури для назви клавіш. Браузери помічають такий текст моношинрим шрифтом.

Приклад використання:
<kbd>Моноширний текст</kbd>

Результат: Моноширний текст

<samp> - використовуєтьс для виводу тексту, котрий явлюється результатом виводу роботи компьбтерної програми або скрипту. Браущери зазвичац відображають текст в даному контейнері за допомогоб моноширного шрифту. За допомогою стилів можна задати власне оформлення.

Приклад використання:
<samp>Моноширний текст</samp>

Результат: Моноширний текст

<pre> - використовується для виведення тексту моноширним шрифтом з усіма пробілами між словами. Даний тег підтримує використання будь-яких допустимих тегів окрім: <big>, <img>, <object>, <small>, <sub> і <sup>.

Приклад використання:
<pre>Моноширний текст</pre>

Результат: 

Моноширний текст

<dfn> - як правило. в документі, коли згадується новий термін, він виділяється курсивом, і дається його визначення. При використанні цього терміна в подальшому, термін буде вже відомим для читача. Даний тег використовується для виділення таких термінів при першому їх появленні в тексті. Браузери відображають вміст даного тега курсивом.

Приклад використання:
<dfn>Новий термін</dfn>

Результат:
Новий термін

<abbr> - тег вказує що послідовність символів являється абревіатурою. За допомогою атрибута title дається розшифровка скорочення, що в свою чергу дозволяє розуміти абревіатуру тим людям, котрі з не знайомі з нею. Може використовуватися для підвищення рейтингу документа.

Приклад використання:
<abbr>Новий термін</abbr>

Результат:
BBC

<q> - використовується для виділення в тексті цитат. Вміст контейнера автоматично відображається в браузері в кавичках.

Приклад використання:
<q>Хочеш змінити світ - заправ постіль</q>

Результат:
Хочеш змінити світ - заправ постіль

<cite> - помічає текст як цитату або виноску, на інший матеріал. Браузер виводить текст контейнера курсивом

Приклад використання:
<cite>Цікава виноска</cite>

Результат:
Цікава виноска

<blockquote> - використовується для виділення довгих цитат в середині окумента.

Приклад використання:
<blockquote>Прям дуууже, дууужее довга цитата</blockquote>

Результат:

Прям дуууже, дууужее довга цитата

<br> - створює розрив для перекладу рядка в тому місці де росташований.

Приклад використання:
<br>Lorem ipsum<br> dolor

Результат:

Lorem ipsum
dolor

<hr> - малює горизонтальну лінію, котра за зовнішнім виглядом залежить від використаних параметрів, а також браузера. Даний тег відноситься до блочних елементів, лінія завжди починається з нового рядка, а після неї всі елементи відображаються на наступному рядку.

Приклад використання:
<hr>Цікава виноска</hr>

Результат:

Цікава виноска

<ul> - створює маркований спискок. Кожен елемент списку повинен починатися з тегу <li>.

Приклад використання:
<ul>
<li>Перший елемент списку</li>
</ul>

Результат:

  • Перший елемент списку

<ol> - встановлює нумерований список. Кожен елемент списка повинен починатися з тега <li>.

Приклад використання:
<ol><li>Елемент нумерованого списку</li>
<li>Елемент нумерованого списку</li>
</ol>

Результат:

  1. Елемент нумерованого списку
  2. Елемент нумерованого списку

<li> - елемент нумерованого списку. Зовнішній тег ul або ol встановлює тип списка - маркований або нумерований

<a href=""> - даний тег являється одним із найважливіших тегів HTML і призначений він для створення посилань. В залежності від наявності атрибута name або href тег <a> встановлює посилання або якір. Якорем - називається закладка всерелині сторінки, яку можна вказати в якості цілі ссилки. Для створення посилання потрібно повідоити браузер, що являється посиланням, а також вказати адресв документа, на який потрібно зробити посилання. В якості занчення атрибута href використовується адреса документа (Url, Universal Resourse Locator, універсальний вказіник ресурсів), на який відбувається перехід.

Приклад використання:
<a href="https://www.google.com/">URL адреса Google</li>

Результат:
URL адреса Google
